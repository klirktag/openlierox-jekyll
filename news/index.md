## OpenLieroX News

<div class="news_item">

OpenLieroX 0.58 rc5 released
{:.news_title}

Posted on 2018-07-30
{:.news_posted}

A new stable release is out! Version 0.58 rc5 adds IPv6 support, shows the death count in games with infinite lives, lets you navigate all menus with the arrow keys for better gamepad support, and fixes desynced shots and the invisible laser in network games. New game options let you mirror the map and make bonuses indestructible. All player skins from the Quality Skins Pack are now included. [Grab it from the downloads page.]({{ '/downloads/' | relative_url }})

</div>

OpenLieroX 0.59 beta10 available for testing
{:.news_title}

Posted on 2012-04-17
{:.news_posted}

The 0.59 beta line brings support for programmable levels and mods (using LUA, based on Gusanos), special light and graphics effects, 3D sounds, sounds for game events, and a single player mode. It is a testing release &ndash; try it out and report any issues. [Download the beta here.]({{ '/downloads/' | relative_url }})
